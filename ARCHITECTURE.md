## Key Design Patterns Used

1. **Provider Pattern** - State management for themes
2. **Single Responsibility** - Each screen handles one feature
3. **Separation of Concerns** - UI, logic, and state separated
4. **Composition over Inheritance** - Widget composition for UI
5. **Responsive Design** - LayoutBuilder for adaptive layouts
6. **Material Design 3** - Modern UI components
