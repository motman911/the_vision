# The Vision - Quick Start Guide

## Installation & Setup

### Step 1: Prerequisites
Make sure you have Flutter installed. Check with:
```bash
flutter doctor
```

### Step 2: Get Dependencies
```bash
cd the_vision
flutter pub get
```

### Step 3: Run the App
```bash
flutter run
```

Or for specific platform:
```bash
flutter run -d chrome      # Web
flutter run -d macos       # macOS
flutter run -d windows     # Windows
flutter run -d linux       # Linux
```

## Troubleshooting

### App won't start
```bash
flutter clean
flutter pub get
flutter run
```

### Theme not changing
- Ensure you're tapping a theme option
- Check if dialog closes after selection

### Navigation issues
- Use back button to return
- Home button always returns to home
