# The Vision - Educational Office Rwanda

Professional Flutter app for "The Vision" office in Rwanda, showcasing educational services, universities, FAQs, and a gallery, with multiple themes for a smooth user experience.

## Features

### Pages
- **Home**: Welcome page with quick links and company overview
- **Services**: Comprehensive list of educational services offered
- **Universities**: Partner universities with detailed information
- **FAQ**: Frequently asked questions and answers
- **Contact**: Contact form and office information
- **Gallery**: Visual showcase of events and activities

### Themes
The app supports multiple themes for user preference:
- Light Theme
- Dark Theme
- Blue Theme
- Green Theme
- Orange Theme

### Responsive Design
The app is fully responsive and works seamlessly on:
- Mobile phones
- Tablets
- Desktop browsers

## Getting Started

### Prerequisites
- Flutter SDK (>=3.0.0)
- Dart SDK (>=3.0.0)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/motman911/the_vision.git
cd the_vision
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```
